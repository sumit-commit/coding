->Nodes are connected in forward direction only

-> In this, each node contains two fields:-
	
(i)  - datafield - for storing data
(ii) - *next -  for storin address of next node


-> It is similar to singly linked list, it has only one change that the last node points to(containes address of) the head node instead of NULL.


class node{
    public:
    int data;
    node* next;
    
    node(int data)
    {
        this->data = data;
        this->next = NULL;
    }

};
